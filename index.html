Conflict and Human Suffering

IronNeverden

Luca Barbati, Lorenzo Negri Ravera, Tommaso Viotti

Overview

This project is concerned with the analysis of data related to political violence and demonstration events, with a specific geographic focus on Europe and Central Asia. The reference dataset is provided by the Armed Conflict Location & Event Data Project (ACLED), a global initiative that collects, curates, and publishes detailed information about political violence, armed conflict, and protest events around the world.

Dataset

The dataset provides an aggregated view. Events are consolidated by temporal period and geographical position, consequently this structure enables the analysis and comparison of spatio-temporal patterns.

Specifically, the dataset provides a multidimensional structure for analysis. Geographically, each record is situated within a macro-area (REGION) and a specific COUNTRY, with events further localized to the primary administrative division (ADMIN1). The temporal axis is defined by the WEEK field, which indicates the start week for data aggregation.

Incident classification is provided at three levels of granularity: EVENT_TYPE (the main category, such as "Protests" or "Battles"), SUB_EVENT_TYPE (the specific classification, like "Peaceful protest"), and DISORDER_TYPE (a broader classification, such as "Political violence"). The primary quantitative metrics are EVENTS, representing the total count of incidents for that grouping, and FATALITIES, the sum of associated victims.

Finally, the dataset includes utility fields for spatial analysis and data linkage: CENTROID_LATITUDE and CENTROID_LONGITUDE provide the geographic centroid coordinates for the ADMIN1 area, POPULATION_EXPOSURE estimates the exposed population, and ID serves as a unique numeric identifier for the area.
