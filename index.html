Project Description

The Armed Conflict Location & Event Data Project (ACLED) is a global initiative that collects, curates, and publishes detailed information about political violence, armed conflict, and protest events around the world. Its primary goal is to provide reliable, real-time data that help researchers, policymakers, journalists, and humanitarian organizations understand where, when, and how violence occurs.

The ACLED database contains millions of records describing events such as battles, attacks on civilians, explosions, and demonstrations. Each event is georeferenced and annotated with attributes such as date, location, type of event, actors involved, and fatalities. The data are compiled by a global network of researchers who monitor local and international sources to ensure accuracy and consistency.

This project makes it possible to analyze spatial and temporal patterns of conflict, track emerging crises, and compare levels of instability across countries and regions. ACLED’s data are freely available and can be visualized using its interactive tools or downloaded for in-depth statistical and geographical analysis.
