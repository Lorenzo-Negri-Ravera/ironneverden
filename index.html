Introduction


This project is concerned with the analysis of data related to political violence and demonstration events, with a specific geographic focus on Europe and Central Asia. The reference dataset is provided by the Armed Conflict Location & Event Data Project (ACLED), a global initiative that collects, curates, and publishes detailed information about political violence, armed conflict, and protest events around the world.

Dataset Overview

The events in the related dataset are consolidated by temporal period and geographical position, consequently this structure enables the analysis and comparison of spatio-temporal patterns.
From a geographical perspective, the data presents a hierarchy from the macro-region to the primary administrative division, while the temporal dimension is defined by a weekly aggregation. To describe the nature of each event, different categories are used, ranging from the most general to the most specific. The main quantitative metrics for analysis include the total number of events and the number of associated fatalities, supplemented by estimates of the exposed population to support more advanced analysis.
